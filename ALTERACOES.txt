🎯 O QUE FOI ADICIONADO
─────────────────────────────────────────────────────────────────────────────

1️⃣ NOVO CAMPO NA CLASSE EstatisticasImagem
   • threshold_crescimento_luz: float
   • Fórmula: média_intensidade + (0.5 × desvio_padrão)
   • Serve como referência para comparar mudanças de luz entre imagens

2️⃣ NOVO MÉTODO: calcular_threshold_crescimento_luz()
   • Calcula automaticamente o threshold para cada imagem
   • Leva em conta variação dos dados (desvio padrão)
   • Executado em processar_imagem()

3️⃣ NOVO MÉTODO: comparar_crescimento_luz()
   • Compara dois períodos (antes e depois)
   • Detecta crescimento: ALTO (>5%), MODERADO (1-5%), ESTÁVEL (<1%), REDUÇÃO (<-1%)
   • Retorna análise com:
     - crescimento_media: diferença absoluta
     - percentual_crescimento: % de mudança
     - threshold_referencia: valor de comparação
     - status_crescimento: emoji descritivo (🔴🟡🟢🔵)
     - crescimento_significativo: se > 0.3×desvio_padrão

📊 EXEMPLO DE USO
─────────────────────────────────────────────────────────────────────────────

from leitura_de_luz import AnalisadorLuzSatelite

analisador = AnalisadorLuzSatelite()

# Processar duas imagens em períodos diferentes
stats_p1 = analisador.processar_imagem(imagem_ano_1)
stats_p2 = analisador.processar_imagem(imagem_ano_2)

# Comparar crescimento de luz
resultado = analisador.comparar_crescimento_luz(stats_p1, stats_p2)

print(f"Status: {resultado['status_crescimento']}")
print(f"Crescimento: {resultado['percentual_crescimento']:.2f}%")
print(f"Threshold ref: {resultado['threshold_referencia']:.2f}")

# Resultado possível:
# Status: 🔴 CRESCIMENTO ALTO
# Crescimento: +8.24%
# Threshold ref: 92.50
